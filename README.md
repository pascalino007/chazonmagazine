# HUMANITÉ+ Magazine

A modern, dark-themed magazine front-end inspired by humanitarian storytelling. Built with **Next.js 14**, **React 18**, **TypeScript**, and **Tailwind CSS**.

## Design

- **Colors**: Deep navy background (`#161c28`, `#1a1b2b`), vibrant orange accent (`#FF6B35`)
- **Sections**: Featured hero, Reportages & Documentaires (video cards), Derniers Articles (with category filters), Témoignages, and Community CTA
- **Responsive**: Mobile-first layout with breakpoints for tablet and desktop

## Getting started

1. **Install dependencies**

   ```bash
   npm install
   ```

2. **Run the development server**

   ```bash
   npm run dev
   ```

   Open [http://localhost:3000](http://localhost:3000) in your browser.

3. **Build for production**

   ```bash
   npm run build
   npm start
   ```

## Project structure

- `src/app/` – Next.js App Router (layout, page, globals)
- `src/components/` – Reusable UI: Header, Footer, FeaturedHero, VideoCard, ArticleCard, CategoryFilters, ReportagesSection, ArticlesSection, TestimonialsSection, CommunityCTA

## Tech stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS 3
